# c-fundamentals_array_algorithms

## Overview

This repository contains basic and intermediate array programs written in C.
The programs focus on common array operations and problem-solving techniques frequently asked in programming interviews, coding practice, and academic labs.

The repository is useful for beginners learning:

Arrays in C
Nested loops
Searching techniques
Sorting algorithms
Duplicate handling
Frequency counting
Basic logic building

## Programs Included
# 1. Array Sorting

Sorts array elements in ascending order using comparison and swapping logic.

# Concepts Used

Nested loops
Swapping
Sorting logic

# 2. Linear Search

Searches for a target element inside an array.

# Concepts Used

Linear traversal
Conditional statements
Search algorithms

# 3. Remove Duplicates from Array

Removes duplicate values from an array by shifting elements.

# Concepts Used

Nested loops
Array shifting
Duplicate detection

# 4. Frequency of Elements

Counts how many times each element appears in an array.

# Concepts Used

Frequency counting
Visited array method
Nested loops
